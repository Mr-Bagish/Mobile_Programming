The Smart Contact Management Android Application (V1) is a mobile application developed using Android Studio for the Mobile Programming course. The main purpose of this app is to help users store, manage, and retrieve contact information easily on their device. It provides a modern and user-friendly interface based on Material Design, ensuring a smooth and attractive user experience.

The application supports full CRUD operations (Create, Read, Update, Delete) for managing contacts. Users can add, edit, delete, and view contact details such as name, phone number, email, and company. All data is stored locally using an SQLite database, and contacts are displayed efficiently using RecyclerView. The app also includes a search feature that allows users to quickly find contacts by name or company.

To improve performance, all database operations are performed asynchronously using Executors, which prevents the app from freezing and ensures a smooth user experience. Overall, this application demonstrates core Android development concepts, including UI design, data storage, and background processing, making it a complete and practical mobile solution.
